===================================
           Qué se hizo?
===================================

Durante este sprint se desarrolló la funcionalidad para gestionar productos dinámicamente en el DOM.

Se implementó:

 * Formulario para agregar productos (nombre y precio).
 * Función en JavaScript para crear elementos dinámicamente.
 * Renderizado automático de los productos en una lista.
 * Flujo de trabajo con ramas feature/* y merge hacia develop.
 * Creación y cierre correcto del Pull Request.

La funcionalidad permite que cada producto ingresado aparezca inmediatamente en pantalla sin necesidad de recargar la página.
